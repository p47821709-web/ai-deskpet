# 桌宠系统设计

## 桌宠生命周期

创建 → AI 生成 → 激活 → 展示 → 交互 → 归档/删除

## 行为状态机

- Idle: 空闲状态（眨眼、环顾四周、伸懒腰）
- Walking: 漫游（随机移动）
- Following: 跟随鼠标
- Playing: 互动玩耍
- Sleeping: 睡眠（低电量/深夜）
- Chatting: 聊天模式

## 情绪系统

情绪通过情绪值 (0-100) 影响动画表现：
- 开心 (happy): 跳跃、旋转
- 难过 (sad): 低头、垂耳
- 生气 (angry): 跺脚
- 惊讶 (surprised): 跳起
- 疲惫 (tired): 打哈欠

## 记忆系统

- 情景记忆: 具体事件的记录
- 语义记忆: 知识类信息
- 程序记忆: 行为模式
- 记忆压缩: 定期合并相似记忆
- 记忆衰减: 低频记忆降权
